# 🧠 Brain Upgrade

Brain Upgrade is a fun math learning website that helps students improve their skills from Grade 1 to Grade 12.

## 🚀 Features
- 🔁 Infinite math questions
- 🎓 Multiple grade levels
- 🏆 Level system
- ❤️ Lives system
- 💰 Coins and shop
- 📊 Score tracking
- 🌐 Leaderboard
- 💾 Save progress

## 🎮 How to Use
1. Enter your name
2. Click "Start"
3. Solve math questions
4. Earn coins and level up
5. Buy lives from the shop

## 🧮 Topics
- Addition
- Multiplication
- Algebra
- Powers
- Problem solving
